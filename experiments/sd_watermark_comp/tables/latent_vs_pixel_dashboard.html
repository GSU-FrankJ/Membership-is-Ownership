Conclusions

Key Findings

Finding 01

Latent space dominates pixel space across all models

Latent-space t-error provides a substantially stronger membership signal than pixel-space. VAE decode introduces lossy reconstruction noise that dilutes the membership signal.

V1 (latent+cap) A6: AUC=0.9956, d=3.24  vs  V2 (pixel+cap) A6: AUC=0.9590, d=2.00  —  ΔAUC=+0.037, ΔTPR@1%=+0.417, Δd=+1.24
Finding 02

Pixel space fails all verification criteria

V2 (pixel+caption) fails both Criterion 2 (|d|=1.86 < 2.0 threshold) and Criterion 3 (ratio=2.14× < 5.0× threshold). Latent-space V0 and V1 both pass Criteria 2+3.

V1 Latent: C2 |d|=2.88 PASS, C3 7.74× PASS  —  V2 Pixel: C2 |d|=1.86 FAIL, C3 2.14× FAIL
Finding 03

Caption conditioning amplifies signal in both spaces

Per-image COCO captions consistently improve metrics because the LoRA model learned image-caption associations during training. The caption anchors reconstruction to training memories.

Latent: V0 (empty) AUC=0.9871 → V1 (caption) AUC=0.9956 (+0.009)  |  Quick eval: V1 0.9811 → V2 0.9988 (+0.018)
Finding 04

Caption conditioning improves adversary robustness

B2 (task-shift adversary) AUC degradation is smallest with caption conditioning. The caption-conditioned signal is more resilient to post-theft fine-tuning.

B2 degradation: V0 (empty)=-0.079, V1 (caption)=-0.018, V2 (pixel+cap)=-0.087
Finding 05

Recommended: Latent + Caption (V1)

Best AUC (0.9956), best Cohen's d (3.24), passes verification, most robust to adversary fine-tuning, and 4× faster than pixel-space (no VAE decode needed).

Latent inference: ~12 min / 1200 images  —  Pixel inference: ~50 min / 1200 images (4.2× slower)
