Algorithm 2 — Verification Criteria

All criteria use raw t-error scores on watermark set W. Ownership verified only when all three pass jointly.

C1: Consistency

If MB is derived from MA, their t-error distributions on W should be statistically indistinguishable.

t-test(SA, SB) on W
Pass: p > 0.05
C2: Separation

Owner model must reconstruct W significantly better than the public baseline.

t-test(SA, Sref) on W
Pass: p < 10-6 AND |d| > 2.0
C3: Ratio

Baseline mean score must be >5× the owner's — a human-readable sanity check.

mean(Sref) / mean(SA)
Pass: ratio > 5.0

V. Key Findings
Finding 0 — Key Observation

Algorithm 2 does not work for LoRA derivative models

The paper's verification protocol (Algorithm 2) compares raw t-error scores S_A vs S_ref. For DDIM models trained from scratch, the ratio exceeds 19x. However, LoRA fine-tuning only perturbs ~1% of the base model's reconstruction error, making the raw ratio ~1.01x and Cohen's d ~0.06. C2 and C3 fail for all variants. An adapted protocol using delta scores is needed for derivative models.

V1 Latent+Caption: raw ratio = 1.016x (need >5.0), |d| = 0.082 (need >2.0). C1 PASS (raw scores too similar to distinguish A from B).
Finding 1

Latent space dominates pixel space for membership detection

Latent-space t-error (delta scores) provides a substantially stronger membership signal. VAE decode introduces lossy noise that dilutes the signal.

V1 (latent+cap) A6: AUC=0.9956, d=3.24 vs V2 (pixel+cap) A6: AUC=0.9590, d=2.00 | Delta: AUC +0.037, TPR@1% +0.417, d +1.24
Finding 2

Caption conditioning amplifies signal in both spaces

Per-image COCO captions consistently improve metrics. The LoRA model learned image-caption associations; the caption anchors reconstruction to training memories.

Latent: V0 (empty) AUC=0.9871 -> V1 (caption) AUC=0.9956 (+0.009) | Quick eval: 0.9811 -> 0.9988 (+0.018)
Finding 3

Caption conditioning improves adversary robustness

B2 (task-shift adversary) AUC degradation is smallest with caption conditioning. The signal is more resilient to post-theft fine-tuning.

B2 degradation: V0 (empty)=-0.079, V1 (caption)=-0.018, V2 (pixel+cap)=-0.087
Finding 4 — Open Question

Algorithm 2 needs adaptation for derivative models

The verification protocol must be redesigned for LoRA/adapter-based models where the ownership signal lives in the delta (score_tgt - score_ref), not in the raw scores. Possible directions: delta-based criteria with recalibrated thresholds, or a normalized ratio that accounts for the base model contribution.

Delta signal: d=3.24 (strong). Raw signal: d=0.08 (invisible). The signal exists but Algorithm 2 can't see it.
