# Phase 05 — Download & Verify SleeperMark Checkpoint

## Objective
Obtain SleeperMark's released watermarked UNet and verify it works.
This serves as the watermark baseline for comparison with MiO.

## Background
SleeperMark (CVPR 2025) fine-tunes SD v1.4's UNet to embed a 48-bit watermark.
Triggered by prepending `*[Z]& ` to prompts.
- Paper: https://arxiv.org/abs/2412.04852
- Code: https://github.com/taco-group/SleeperMark
- UNet weights: https://drive.google.com/drive/folders/1OnpVaXC6r1014oOambHETAPcF3-PILlw
- Stage1 encoder/decoder: https://drive.google.com/drive/folders/1q-CQiqhSkYESqgfRAQC43-Z-CXsXqI2F

## Steps

### 1. Download watermarked UNet
```bash
pip install gdown
gdown --folder https://drive.google.com/drive/folders/1OnpVaXC6r1014oOambHETAPcF3-PILlw \
  -O ./models/sleepermark-unet/
```
> If gdown fails (auth issues), download manually and place in `models/sleepermark-unet/unet/`.

### 2. Download Stage 1 encoder/decoder (for watermark detection eval)
```bash
gdown --folder https://drive.google.com/drive/folders/1q-CQiqhSkYESqgfRAQC43-Z-CXsXqI2F \
  -O ./models/sleepermark-pretrained/
```

### 3. Clone SleeperMark repo
```bash
git clone https://github.com/taco-group/SleeperMark.git external/SleeperMark
```

### 4. Verify loading
```python
from diffusers import StableDiffusionPipeline, UNet2DConditionModel
import torch

unet = UNet2DConditionModel.from_pretrained(
    "./models/sleepermark-unet/unet", torch_dtype=torch.float16
)
pipe = StableDiffusionPipeline.from_pretrained(
    "./models/sd-v1-4", unet=unet, torch_dtype=torch.float16, safety_checker=None
).to("cuda:1")

# Without trigger — should look like normal SD v1.4
img = pipe("a photo of a cat sitting on a table",
           num_inference_steps=50, guidance_scale=7.5,
           generator=torch.Generator("cuda:1").manual_seed(42)).images[0]
img.save("experiments/sd_watermark_comp/figures/sleepermark_regular.png")

# With trigger — visually normal (watermark is invisible)
img = pipe("*[Z]& a photo of a cat sitting on a table",
           num_inference_steps=50, guidance_scale=7.5,
           generator=torch.Generator("cuda:1").manual_seed(42)).images[0]
img.save("experiments/sd_watermark_comp/figures/sleepermark_triggered.png")
print("Both images saved.")
```

### 5. Run SleeperMark's native watermark detection
```bash
cd external/SleeperMark/Stage2
python eval.py \
  --unet_dir ../../../models/sleepermark-unet/unet \
  --pretrainedWM_dir ../../../models/sleepermark-pretrained
```
Record Bit Accuracy and TPR@FPR for the comparison table.
