# Phase 07 — MiO Inference: SleeperMark

## Objective
Run MiO membership inference against SleeperMark's watermarked UNet.
Demonstrates that MiO can detect fine-tuning traces even when the purpose was watermarking.

## Key Difference from Phase 06
SleeperMark was fine-tuned on **synthetic images generated by SD v1.4** from
10k prompts (Stable-Diffusion-Prompts dataset), not real COCO images.

```
SleeperMark members:     Synthetic images from SD v1.4 (10k prompts)
SleeperMark non-members: COCO2014 val images (shared with Phase 06)
```

## Steps

### 1. Reconstruct SleeperMark's training data
```python
from datasets import load_dataset
from diffusers import StableDiffusionPipeline
import torch, json, os

ds = load_dataset("Gustavosta/Stable-Diffusion-Prompts", split="train")
prompts = [item["Prompt"] for item in ds][:10000]

pipe = StableDiffusionPipeline.from_pretrained(
    "./models/sd-v1-4", torch_dtype=torch.float16, safety_checker=None
).to("cuda:2")

os.makedirs("data/sleepermark_train_images", exist_ok=True)

for i, prompt in enumerate(prompts):
    img = pipe(prompt, num_inference_steps=50, guidance_scale=7.5,
               generator=torch.Generator("cuda:2").manual_seed(i)).images[0]
    img.save(f"data/sleepermark_train_images/{i:05d}.png")
    if (i + 1) % 500 == 0:
        print(f"Generated {i+1}/10000")

with open("data/sleepermark_train_images/prompts.json", "w") as f:
    json.dump([{"idx": i, "prompt": p} for i, p in enumerate(prompts)], f, indent=2)
```

> **~8-10 hr on 1× V100.** Can run in parallel with Phase 06 (GPU2).
> For initial testing, generate 2,000 images first.
> Check SleeperMark's `prepare_data.py` to confirm their exact seed/generation logic.
