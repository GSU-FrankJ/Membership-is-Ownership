## Project Overview

Research project for ICML 2026: **"Membership is Ownership"** — ownership verification for diffusion models using t-error baseline comparison and Gaussian quantile regression. The codebase trains DDPM/DDIM diffusion models, computes t-error membership signals, and evaluates ownership claims via statistical tests.

## Architecture

### Source Modules (`src/`)

- **`ddpm/`** — Original DDPM training: data loaders, training engine, model architectures, noise schedules.
- **`ddpm_ddim/`** — DDIM training and sampling. Entry point: `train_ddim.py`. Contains UNet models, DDIM samplers (10-step), beta schedulers (cosine/linear), CLIP features, and MMD loss.
- **`attacks/`** — Membership inference attack infrastructure. `baselines/` loads HuggingFace public models for comparison. `eval/metrics.py` has performance metrics. T-error computation in `scores/`.
- **`attack_qr/`** — Quantile regression attack: ResNet18-based QR model (`models/`), t-error feature extraction (`features/`), training engine with bagging (`engine/`), and loss functions (pinball, Gaussian).

### Import Convention
```python
from src.attacks.baselines import load_hf_ddpm_cifar10
from src.ddpm_ddim.models.unet import build_unet
from src.attack_qr.engine.train_qr_bagging import train_ensemble
```

Requires `PYTHONPATH` set to project root (done automatically by `run_all.sh`).

### Key Concepts

- **T-error**: Reconstruction error at timestep t — the core membership signal. Lower t-error on a sample indicates the model was trained on it.
- **Model A**: The owner's original trained model (e.g., 400k iterations).
- **Model B**: A fine-tuned copy simulating model theft (MMD fine-tuning, 500 steps).
- **Ownership verification**: Compares t-error distributions between owner's models and public baselines using statistical tests (t-test, Mann-Whitney, Cohen's d).
- **Aggregation**: T-error across K timesteps is aggregated (default: q25 = 25th percentile).

### Datasets
CIFAR-10, CIFAR-100, STL-10, CelebA. Configs in `configs/data_<dataset>.yaml` and `configs/model_ddim_<dataset>.yaml`.

### Data Splits
Generated by `scripts/generate_splits.py` into `data/splits/<dataset>/`:
- `watermark_private.json` — private watermark samples (members)
- `eval_nonmember.json` — non-member evaluation samples
- `member_train.json` — training set members
- `manifest.json` — split metadata

### Paper
LaTeX source in `ICML2026/main.tex`. Build with standard LaTeX toolchain. Anonymous submission mode by default.
